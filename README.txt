PROJECT TITLE: Guessing Game

VERSION or DATE: 21/05/2022

AUTHORS: Ivan Potgieter

USER INSTRUCTIONS: 
Enter a number in the textbox. Once a number is entered, click the "Check match" button. The display pane returns a message containing information on your guess for example, that it was too high. You can adjust your guess accordingly and guess again. The program will display a message  when you have guessed the correct number.
